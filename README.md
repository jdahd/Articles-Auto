# Article Purifier

一键下载公众号文章并转换为Markdown/WORD的桌面工具，支持图片本地化

## 功能特点

\- 🚀 单篇/批量下载公众号文章

\- 📝 自动转换为 Markdown 格式，排版不变

\- 🖼️ 文章内图片自动下载到本地，不丢失

\- 📄 支持导出 PDF/Word 格式（如果你的工具支持就留，不支持就删）

\- 🔗 可同步到 Obsidian/Notion 知识库（如果你的工具支持就留）

\- 🎨 简单好用的图形界面，新手也能上手



\## 🛠️ 安装使用

\### 1. 环境要求

\- Python 3.8 及以上版本（建议 3.10）



\### 2. 安装依赖

打开命令行/终端，进入项目文件夹，执行：

```bash

pip install -r requirements.txt

\### 使用步骤

* 运行程序后，粘贴公众号文章链接到输入框
* 选择保存格式（Markdown/PDF）和保存路径
* 点击「开始下载」，等待完成即可
